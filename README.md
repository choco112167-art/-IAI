IAI - AI 선택비서
=================

상황만 편하게 한 줄로 쓰면, 지금 하고 싶은 일에 잘 맞는 AI 도구를 골라서 추천해주는 작업 상황 분류 기반 AI 추천 서비스 (PoC)입니다.

## 1. 프로젝트 개요
----------------

- 사용자가 한국어로 입력한 자연어 문장을 간단한 규칙 기반 분류기로 작업 카테고리(회의/문서/이미지/영상/코드/PPT/리서치)로 분류합니다.
- 각 카테고리별로 미리 정리해 둔 대표 AI 도구 리스트에서 3~4개 정도를 골라 추천합니다.
- 프론트엔드 없이, Colab/터미널에서 텍스트 입출력으로만 동작하는 가벼운 프로토타입입니다.


## 2. 실행 방법
------------

1. 이 레포지토리를 클론하거나, `iai_poc.py` 파일을 Colab에 업로드합니다.
2. Colab에서 해당 파일 내용을 하나의 코드 셀에 붙여 넣고 실행합니다.
3. 셀 실행 후 프롬프트가 나오면, 현재 하고 싶은 작업 상황을 한국어로 자연스럽게 입력합니다.

예시 입력
- 팀플 회의 녹음해 둔 거 대충 정리해서, 뭐 해야 할지만 알고 싶어
- 졸업논문 관련 선행연구 논문 좀 찾아보고 정리해야 돼
- 유튜브 쇼츠 영상이랑 썸네일 같이 만들어야 하는데, 어떤 AI 쓰면 좋을지 모르겠어


## 3. 사용 예시 (콘솔 출력)
------------------------

```bash
===== IAI - AI 선택비서 (규칙 기반 PoC) =====
지금 뭐 하려고 하는지 편하게 한 줄로 써주세요 :)
> 팀플 회의 녹음해 둔 거 대충 정리해서, 뭐 해야 할지만 알고 싶어

[판단된 작업 카테고리]
- meeting_summary

[추천 AI 도구]

1. 클로바노트
   - 설명: 한국어 회의 녹음 내용을 자동으로 받아 적고, 중요한 내용 위주로 요약해 주는 도구
   - 강점: 국내 회의 환경(한국어 음성)에 최적화되어 있어서 팀플/스터디 회의 정리에 쓰기 좋음

2. Notion AI
   - 설명: 노션에 적어 둔 회의록이나 메모를 한 번에 요약하고, 할 일 목록 형태로 정리해 주는 도구
   - 강점: 이미 노션을 쓰고 있다면 별도 셋업 없이 바로 회의 내용 정리·정돈에 활용 가능

3. Notta
   - 설명: 화상회의를 자동으로 녹음·전사하고, 핵심 내용만 골라 보여주는 회의 어시스턴트
   - 강점: Zoom·Google Meet 등 주요 회의 플랫폼과 연동되어 온라인 회의가 많을 때 유용함
